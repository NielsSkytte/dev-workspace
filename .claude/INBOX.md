# Workspace Inbox

A single, low-friction place to **park anything before it gets forgotten** — known tasks, half-formed ideas, open questions, plain to-dos. Generic or project-specific, doesn't matter at capture time.

**Capture:** type `/park <whatever>` from any session. No need to classify, route, or format — just dump it.
**Process:** items get triaged later (eventually on a daily schedule) — classified, routed to a project, and shaped into something actionable. For now, triage is manual / on request.

Each item: `- [ ] YYYY-MM-DD — the thing`. Check it off (`[x]`) or delete when done. Don't worry about order or grouping — the sweep reorganizes.
