# Workspace TODO

The workspace's single, low-friction **action capture** — the ICOR *Input* stage. Capture anything before it gets forgotten: known tasks, half-formed ideas, open questions, plain to-dos. No need to decide where it belongs at capture time.

> Distinct from a project's `INBOX.md` (its curated cross-project knowledge feed via `/brief`). This is *your action list*; that is *a project's incoming knowledge*.

**Capture:** type `/todo <whatever>` from any session. No need to classify, route, or format — just dump it.
**Process:** items get triaged later (eventually on a daily schedule) — classified, routed to a project, and shaped into something actionable. For now, triage is manual / on request.

Each item: `- [ ] YYYY-MM-DD — the thing`. Check it off (`[x]`) or delete when done. Don't worry about order or grouping — the sweep reorganizes.
